## PickHue 2.0.2 — Polish

A small follow-up to 2.0.1: custom swatch tooltips, unified icons, clearer export options, and reliability fixes.

**Free. No accounts. No tracking. Nothing leaves your machine.**

---

### What's new in 2.0.2

- Custom PickHue-styled color tooltips on swatches (with delayed show so remove stays easy)
- Hover-to-remove restored on Recent Colors
- Unified outline icons; Rename / Export / Move / Delete icons in the palette row menu
- Export menus include hex list, CSS variables, Adobe ASE, and PickHue JSON
- Empty palette “Select Color” text links straight into the picker
- Scroll, toast-menu, and storage reliability polish

---

### Install (developer)

1. Download **pickhue-2.0.2.zip** below
2. Unzip the archive
3. Open `chrome://extensions` or `brave://extensions`
4. Enable **Developer mode** → **Load unpacked** → select the unzipped folder
5. Pin PickHue from the toolbar puzzle menu

**Build from source:** `git checkout 2.0.2 && npm install && npm run build` → load `dist/`

Chrome Web Store users on 2.0.1 will receive this update after review.

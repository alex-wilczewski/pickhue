## PickHue 2.0.3 — Theme & save flow

Picker overlays and Save to palette behave more consistently with your theme and intent.

**Free. No accounts. No tracking. Nothing leaves your machine.**

---

### What's new in 2.0.3

- Loupe instructions and copy toast follow your theme (system / light / dark)
- Light-mode overlays use the same subtle translucency as dark
- Save to palette: palettes stay greyed until you select a recent color; gear menu shows disabled instead of disappearing
- Palette pick animation no longer clips in the scroll list
- “Add colors” confirm uses accent green; palettes unlock once a swatch is selected

---

### Install (developer)

1. Download **pickhue-2.0.3.zip** below
2. Unzip the archive
3. Open `chrome://extensions` or `brave://extensions`
4. Enable **Developer mode** → **Load unpacked** → select the unzipped folder
5. Pin PickHue from the toolbar puzzle menu

**Build from source:** `git checkout 2.0.3 && npm install && npm run build` → load `dist/`

Chrome Web Store users on 2.0.2 will receive this update after review.

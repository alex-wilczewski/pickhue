## PickHue 2.0.1 — Polish

A small follow-up to the 2.0 palettes release: clearer editing cues, smoother palette list browsing, and a few UI fixes.

**Free. No accounts. No tracking. Nothing leaves your machine.**

---

### What's new in 2.0.1

- Palette title looks editable from the start
- Saved Palettes list scrolls after five rows, with a soft fade instead of a harsh clip
- **Move** in the palette row menu — drag one palette to reorder the list
- Preview shows at most four swatches
- Filled delete/settings icons and other small visual fixes
- Recents picker and scroll containment polish

---

### Install (developer)

1. Download **pickhue-2.0.1.zip** below
2. Unzip the archive
3. Open `chrome://extensions` or `brave://extensions`
4. Enable **Developer mode** → **Load unpacked** → select the unzipped folder
5. Pin PickHue from the toolbar puzzle menu

**Build from source:** `git checkout 2.0.1 && npm install && npm run build` → load `dist/`

Chrome Web Store users on 2.0.0 will receive this update after review.
